BootstrapSideNav

This navigation for Twitter Bootstrap works as an addition to the existing
<ul class="nav nav-tabs nav-stacked"></ul> list. On the left side you can see an example.

Usage

Basic Example

  <ul class="nav nav-tabs nav-stacked nav-side">
    <li>
      <a href="/">
        <div class="nav-title">Home</div>
        <i class="icon-user nav-icon"></i>
      </a>
    </li>
  </ul>

See the home menu item in the list on the left.

No Link

  <ul class="nav nav-tabs nav-stacked nav-side">
    <li class="empty">
      <div class="nav-title">Home</div>
      <i class="icon-user nav-icon"></i>
    </li>
  </ul>

See the search menu item in the list on the left.

Header

  <ul class="nav nav-tabs nav-stacked nav-side">
    <li class="nav-header">Title</li>
  </ul>

See the two headers in the list on the left.

Dropdown Menu

  <ul class="nav nav-tabs nav-stacked nav-side">
    <li class="dropdown">
      <a class="dropdown-toggle" data-toggle="dropdown" href="#">
        <div class="nav-title">Settings <b class="caret"></b></div>
        <i class="icon-wrench nav-icon"></i>
      </a>
      <ul class="dropdown-menu">
        <li>
          <a href="#">
            <div class="nav-title">Edit</div>
            <i class="icon-edit nav-icon"></i>
          </a>
        </li>
      </ul>
    </li>
  </ul>

See the settings menu item in the list on the left.

Additional Settings

Lighten

For the menu to be lightend when it's inactive you can just add the class lighten to the list.

example

Hidden-Icons

For the icons to disapper when the menu slides out you can just add the class hidden-icons to the list.

example

Front-Icons

For the icons to got to the front when the menu slides out you can just add the class front-icons to the list.
